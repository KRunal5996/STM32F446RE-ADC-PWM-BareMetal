# STM32F446RE ADC Controlled PWM Bare-Metal Project

This project demonstrates ADC-controlled PWM generation on STM32F446RE using bare-metal register-level programming in Embedded C without HAL libraries.

Features:
- ADC driver
- PWM generation using TIM2
- UART communication
- SysTick delay
- GPIO configuration
- Register-level peripheral programming

Microcontroller:
STM32F446RE

Development Environment:
STM32CubeIDE

Language:
Embedded C
